# 🐉 Chest Companion –Tester

**Development & Testing Build**

Welcome to the private testing version of **Chest Companion**, a free companion app for the War Dragons community.

This repository is used to trial new features, identify bugs, and improve the app before updates are released to the public beta.

---

## 🧪 Purpose

Eff Tester is an experimental build where new functionality is tested before release.

Features may be:
- Incomplete
- Changed frequently
- Broken from time to time (hopefully not too often 😅)

---

## ✅ Current Features

- Gold Chest Predictor
- Platinum Chest Predictor
- Chest Reward Tracker
- Sequence Detection
- CSV Import
- CSV Export
- Event Statistics
- Local Save Support

---

## 🚧 Currently in Development

- Improved Sequence Matching
- Better Error Messages
- Double Armory Support
- Reward Planner
- Future Chest Predictions
- Developer Testing Tools
- UI Improvements
- Performance Optimisation

---

## 🐛 Reporting Bugs

If you find a bug, please provide:

- Screenshot
- Event
- Chest Type
- Rewards entered
- What happened
- What you expected to happen

The more information provided, the easier it is to reproduce and fix.
